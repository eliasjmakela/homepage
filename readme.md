## Mitkä asiat teit

Tein pääsivun ja projektit-alasivun. Blogi, CV ja projektikohtaiset alasivut jäivät tekemättä. Päätin mieluummin tehdä pääsivun+yhden alasivun niin hiottuna kuin pystyin, kuin tehdä monta vähän sinne päin. Aion jatkaa tämän sivun työstämistä ja hostata sen omassa domainissa kun se on valmis.

## Mikä tehtävässä oli haastavaa? Muuttuiko sivusto matkan varrella alkuperäisestä suunnitelmastasi?
Layoutin ja responsiivisuuden säätäminen oli varmasti vaikeinta, sekä itse CSS-väännön että design-ratkaisujen tekemisen kannalta. Sivu muuttui paljon alkuperäisestä, mutta jotain Figma-leiskojen hengestä siihen jäi. Mielestäni työstön aikana sivusta hioutui paljon parempi kuin mitä se suunnitelmassa oli.
